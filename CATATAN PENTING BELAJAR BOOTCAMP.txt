- Position CSS
1) Static   : artinya adalah dengan default html layoutnya atau mengikuti aturan html.
2) Relative : kalau relative itu akan bermula dari aturan html/layout nya.jadi ketika misalnya digeser posisi awal nya akan tetap ada atau bertaha.
	      ibaratkan jasad kalau rohnya digeser maka jasadnya tetep di posisi awal.
3) Absolute : kalau absolut kebalikan dari relative,jadi parentnya atau jasadnya adalah ruangan.maksud ruangan adalah pembungkus/div/container atau apaupun.
4) Fixed    : kalau fixed maka akan mengikuti posisinya jika misalkan kita scroll kebawah.

- CSS FONT SIZE
1) 1em sama dengan 16px.
2) Dibandingkan px,% dan em,disarankan menggunakan rem karena tidak akan terikat dengan induk dan tidak rawan kesalahan,dan juga responsive.
3) Cara menghitung dari pixel ke rem : misalkan kita ingin 40px.nah 1 em = 16px,maka dari itu tinggal dibagi saja : 40/16 = 2.5rem

- FLOAT AND CLEAR
1) Jadi kalau suatu pembungkus mempunyai dua objek dan objeknya itu ada di atas dan keduanya dibawah maka :
   1a) Jika objek kedua kita kasih float:left misalkan.itu akan disebelah kanannya objek pertama.
   1b) Kalau dikasih float right maka kebalikannya.
   1c) Float itu ibaratkan bakal ikut aliran,nah sumber alirannya adalah dari objek pertama.
2) Kalau clear itu kaya anti float,jadi misalkan tadi dikasih float left,lalu ada satu elemen di objek tersebut kita kasih clear:left; maka akan kebawah.
3) Gunakan float jika perlu saja,dan jangan terlalu mengandalkannya sebagai positioning.
